# IDZ2_progect
МЯСО&amp;ТЕСТО — учебный проект сайта пельменной. Это адаптивный лендинг, разработанный в рамках практики HTML, CSS и JavaScript.


## Функциональность
- Адаптивная верстка (mobile / tablet / desktop)
- Слайдер интерьера
- FAQ-аккордеон
- Бургер-меню
- Модальные окна
- Форма бронирования с валидацией
- Маска ввода телефона
- Интерактивное меню блюд
- SEO-оптимизация (meta tags, Open Graph, Schema.org)


## Технологии
- HTML5
- CSS3
- JavaScript


## Структура проекта
├── index.html          # Главная HTML-страница
├── css/
│   └── styles.css      # Все стили проекта
├── js/
│   └── script.js       # Весь JavaScript-код
├── assets/
│   ├── fonts/          # Шрифты
│   ├── icons/          # Иконки 
│   ├── images/         # Изображения 
│   └── videos/         # Видеофайлы
├── robots.txt          # Инструкции для поисковых роботов
├── sitemap.xml         # Карта сайта
└── README.md           # Описание проекта


## Статус проекта
Учебный проект (не коммерческий)
