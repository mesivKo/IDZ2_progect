# IDZ2_progect
МЯСО&amp;ТЕСТО — учебный проект сайта пельменной. Это адаптивный лендинг, разработанный в рамках практики HTML, CSS и JavaScript.


## Функциональность
- Адаптивная верстка (mobile / tablet / desktop)
- Слайдер интерьера
- FAQ-аккордеон
- Бургер-меню
- Модальные окна
- Форма бронирования с валидацией
- Маска ввода телефона
- Интерактивное меню блюд
- SEO-оптимизация (meta tags, Open Graph, Schema.org)


## Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, адаптивность)
- Vanilla JavaScript (ES6+)
